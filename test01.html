HTML

HTML abreviação para a expressão inglesa HyperText Markup Language,
que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem
de marcação utilizada na construção de páginas na Web. Documentos HTML
podem ser interpretados por navegadores. A tecnologia é fruto da
junção entre os padrões HyTime e SGML.

Como a linguagem HTML funciona

Com o HTML, é possível definir a estrutura, o posicionamento dos elementos,
a divisão de uma página em blocos visuais e combinar texto, imagens e vídeos
para aplicações web.
Essa linguagem é o componente mais importante de qualquer site, já que sem
ela os outros elementos adicionados com programação, como a ação resultante
de clicar em um botão, não têm encaixe.

HTML: Linguagem de Marcação de Hipertexto

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição
em um navegador da Web. A marcação HTML inclui "elementos" especiais, veja mais
Um elemento HTML é separado de outro texto em um documento por "tags", que consistem
no nome do elemento entre "<" e ">". O nome de um elemento dentro de uma tag é insensível
a maiúsculas e minúsculas.
